# Jonasnygaard.com

[https://www.jonasnygaard.com](https://www.jonasnygaard.com)

Personal portfolio website of **Jonas Nygaard** — frontend developer & designer.

**Tech stack:** Next.js, TypeScript, Tailwind CSS, Framer Motion, React Three Fiber (R3F)

**Features:** Responsive design, interactive 3D elements, smooth animations, project showcase.

**Run locally:**

```bash
git clone https://github.com/YOUR_USERNAME/jonasnygaard.com.git
cd jonasnygaard.com
npm install
npm run dev
```
